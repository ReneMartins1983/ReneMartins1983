## 🚀 Projetos pessoais

> Projetos abertos que construí para explorar tecnologias e demonstrar na prática
> (meu trabalho principal é em código privado).

### 🧩 [CVForge](https://github.com/ReneMartins1983/CVForge) — gerador de currículos
Monte currículos profissionais com **prévia ao vivo**, 11 modelos (com/sem foto) e exportação em PDF.
**Stack:** Laravel 12 · Blade · Alpine.js · MySQL · Docker
🌐 **[Demo ao vivo](https://cvforge-zv12.onrender.com)** · 📂 [Código](https://github.com/ReneMartins1983/CVForge)

### 📡 [GeoPulse](https://github.com/ReneMartins1983/GeoPulse) — painel de frota em tempo real
Dashboard com **mapa interativo (Leaflet)** e **gráficos (Chart.js)** consumindo uma **API REST**.
**Stack:** Laravel 12 (API REST) · Vue 3 (SPA) · Leaflet · Chart.js · Docker
🌐 **[Demo ao vivo](https://geopulse-crsn.onrender.com)** · 📂 [Código](https://github.com/ReneMartins1983/GeoPulse)

### 👥 [Clientela](https://github.com/ReneMartins1983/Clientela) — mini-CRM
Gestão de clientes e atendimentos: **funil (kanban) com arrastar-e-soltar**, follow-ups, anexos e painel.
**Stack:** Laravel 12 · Livewire 3 · Tailwind · MySQL · Docker
🌐 **[Demo ao vivo](https://clientela.onrender.com)** _(demo@clientela.app / password)_ · 📂 [Código](https://github.com/ReneMartins1983/Clientela)

> Juntos, os três cobrem as principais abordagens de front-end do ecossistema Laravel
> — **Blade + Alpine**, **Vue 3 + API REST** e **Livewire** — todos dockerizados, testados e com CI.
